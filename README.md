# Games_By_CRUD_API
#  Games Information API (Flask + Vercel)

A simple backend project built using **Flask** that serves structured game data from a local JSON file and displays it through a clean web interface. Deployed using **Vercel serverless platform**.

---

##  Live Demo

👉 https://your-project.vercel.app/

---

##  Features

-  Reads game data from a local `games.json` file  
-  Displays data in a clean HTML format  
-  Lightweight Flask backend  
- Deployed on Vercel (serverless)  
- Beginner-friendly backend architecture  

---

##  What This Project Does

This project loads structured game information from a JSON file and renders it dynamically on a web page.

It helps understand:
- Backend routing with Flask  
- JSON data handling in Python  
- Server-side rendering  
- Deployment on cloud platforms
